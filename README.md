# Task Dashboard 🚀

A modern, responsive, and beautiful Kanban-style task management dashboard built with React and TypeScript.

## ✨ Features

- 📋 **Kanban Board Layout:** Organize tasks into 'To Do', 'In Progress', and 'Done' columns.
- ➕ **Task Management:** Easily create, edit, and delete tasks.
- 🎨 **Beautiful UI:** A sleek dark-mode design with smooth gradients, glassmorphism, and micro-animations.
- 🏷️ **Priority Levels:** Assign High, Medium, or Low priorities to your tasks with visual indicators.
- 💾 **Local Storage Persistence:** Your tasks are automatically saved in the browser, so you never lose your data even if you refresh the page.
- ⚡ **Lightning Fast:** Built with Vite for rapid development and optimized production builds.

## 🛠️ Tech Stack

- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Vanilla CSS (CSS Variables, Flexbox/Grid)
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Moustafahany90/task-dashboard.git
   ```

2. Navigate into the project directory:
   ```bash
   cd task-dashboard
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173` to view the app!

## 📂 Project Structure

- `src/components`: Reusable UI components.
- `src/context`: React Context for state management (TaskContext).
- `src/features`: Core feature components like the Task List, Task Item, and Task Form.
- `src/types`: TypeScript interfaces for task types.
- `src/App.css` & `src/index.css`: Global styles and UI tokens.
